产品名称: Uviewer - 现代图像与小说阅读器

简短说明:
一款支持图像、文本、PDF 和 EPUB 的多功能查看器，支持青空文库(竖排文字)和压缩包直接浏览。基于 WinUI 3 的现代高性能体验。

详细说明:
Uviewer 是一款面向 Windows 的现代高性能综合查看器，基于 WinUI 3 和 .NET 10 构建。它可流畅处理图像、文本、PDF 和 EPUB 文档，并提供压缩包内图像直接浏览、双页并排模式，以及完整的青空文库竖排文字渲染等高级功能。

Uviewer 还支持 Markdown、智能编码检测和 WebDAV 连接，可让你的数字阅读与浏览流程更加顺畅。

主要功能:

🖼️ 图像与 PDF 查看
- 广泛格式支持: PDF、JPEG、PNG、GIF、BMP、TIFF、ICO、SVG、WebP、AVIF、JXL 和动态 WebP。
- 高性能渲染: 使用 Win2D (Direct2D)，缩放和平移更加流畅。
- 智能缩放: 支持自定义缩放级别(0.1x 到 10x)、适应窗口和实际大小。
- 工具: 锐化滤镜、双页并排模式、匹配控制方向(支持从右到左导航)和快速导航浮层。

📝 文本与小说阅读
- 高级文本引擎: 大型文本文件也能快速加载。
- 青空文库支持: 原生渲染 Ruby 注音、强调、缩进和插图。
- Markdown 支持: 自定义渲染表格与代码块。
- 智能编码检测: 支持 UTF-8、EUC-KR、Shift-JIS 等。
- 竖排模式: 以像素级布局精度提供完整的竖排阅读体验。
- 导航: 支持目录提取、PDF 书签和跳转到行。

📖 EPUB 阅读器
- 完整 EPUB 支持: 基于页面的导航和章节跟踪。
- 目录导航: 快速在章节之间跳转。
- 竖排支持: 以传统阅读方式查看 EPUB。

📂 文件管理
- 压缩包支持: 无需解压即可直接查看 .zip、.rar、.7z 等压缩包中的图像。
- 整理功能: 管理收藏和最近文件。
- WebDAV 支持: 直接访问远程服务器上的图像。

🌍 本地化
- 多语言支持: 英语、韩语、日语、简体中文、繁体中文和越南语。

功能:
- 在一个应用中无缝查看图像、文本、PDF 和 EPUB。
- 无需解压即可直接查看 ZIP/7z/RAR 压缩包中的图像。
- 支持 Ruby 注音的精美青空文库(竖排文字)渲染。
- 硬件加速图像渲染，提供流畅性能。
- 文本文件智能编码自动检测。
- 支持 WebDAV 远程阅读与浏览。
- 现代 Windows 11 风格界面，支持深色/浅色模式。
