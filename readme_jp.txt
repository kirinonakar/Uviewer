【ソフト名】 Uviewer
【概    要】 Windows 向けの多機能・統合ビューアです。画像、テキスト、EPUB の閲覧をシームレスに行うことができます。WinUI 3 と .NET 10 をベースにしており、高いパフォーマンスを提供します。
【主な機能】
・画像表示：JPEG, PNG, WebP, Archive(ZIP/7z/RAR)内の直接表示
・テキスト表示：青空文庫形式（縦書き対応）、Markdown形式
・EPUB閲覧：ページナビゲーション、縦書き対応
・WebDAV接続：リモートサーバー上のファイルを直接閲覧

【動作環境】 Windows 10 (Version 19041+) または Windows 11
【インストール】
1. 配布されたZIPファイルをダウンロードします。
2. 任意の場所に解凍します。
3. `Uviewer-v1.1.8-x64.exe` を実行してください。
【アンインストール】
・Windowsの「設定 > アプリ > インストールされているアプリ」(またはコントロールパネルの「プログラムの追加と削除」)からアンインストールを行ってください。
