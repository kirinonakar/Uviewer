제품 이름: Uviewer - 현대적인 이미지 및 소설 뷰어

요약 설명:
이미지, 텍스트, PDF, EPUB를 지원하는 다기능 뷰어. 아오조라 문고(세로쓰기) 및 압축 파일 직접 보기 지원. WinUI 3 기반의 현대적이고 강력한 성능.

상세 설명:
Uviewer는 WinUI 3 및 .NET 10 기반으로 구축된 현대적이고 고성능의 Windows용 통합 뷰어입니다. 이미지, 텍스트, PDF, EPUB 문서를 매끄럽게 처리하며, 압축 파일 내 이미지 직접 보기, 2페이지 보기 모드, 아오조라 문고(Aozora Bunko)의 완벽한 세로쓰기 렌더링 등 고급 기능을 제공합니다.

유연하게 설계된 Uviewer는 Markdown 지원, 스마트 인코딩 감지, WebDAV 연결 기능을 갖추고 있어 디지털 독서 및 감상 워크플로우를 최적화합니다.

주요 기능:

🖼️ 이미지 및 PDF 감상
- 광범위한 포멧 지원: PDF, JPEG, PNG, GIF, BMP, TIFF, ICO, SVG, WebP, AVIF, JXL 및 애니메이션 WebP 지원.
- 고성능 렌더링: Win2D(Direct2D) 기술을 사용하여 부드러운 확대/축소 및 이동 가능.
- 스마트 스케일링: 사용자 지정 배율(0.1x ~ 10x), 창 맞춤, 실제 크기 보기 지원.
- 도구: 선명도 필터, 2페이지 보기 모드, 컨트롤 방향 일치(우측에서 좌측으로 이동 지원), 빠른 탐색 오버레이.

📝 텍스트 및 소설 감상
- 고급 텍스트 엔진: 대용량 텍스트 파일의 즉각적인 로딩.
- 아오조라 문고 지원: 루비(후리가나), 강조, 들여쓰기, 이미지의 네이티브 렌더링 지원.
- Markdown 지원: 표, 코드 블록을 포함한 사용자 정의 Markdown 렌더링.
- 스마트 인코딩 감지: UTF-8, 확장 완성형(EUC-KR), Shift-JIS, 조합형 등을 자동으로 감지하여 처리.
- 세로쓰기 모드: 픽셀 단위의 정확한 레이아웃으로 완벽한 세로쓰기 환경 제공.
- 탐색: 목차(TOC) 추출, PDF 북마크 및 특정 줄 이동 지원.

📖 EPUB 리더
- 완전한 EPUB 지원: 챕터 추적 기능이 포함된 페이지 기반 탐색.
- 목차 탐색: 목차를 통한 빠른 이동 지원.
- 세로쓰기 지원: 전통적인 독서 경험을 위한 EPUB 세로쓰기 모드 지원.

📂 파일 관리
- 압축 파일 지원: .zip, .rar, .7z 등의 압축 파일 내 이미지를 압축 해제 없이 직접 감상.
- 관리 기능: 즐겨찾기 및 최근 파일 관리.
- WebDAV 지원: 원격 서버의 이미지를 직접 감상 가능.

🌍 현지화
- 다국어 지원: 한국어, 영어, 일본어 자동 감지 및 지원.

기능 목록:
- 이미지, 텍스트, PDF, EPUB를 하나의 앱으로 매끄럽게 감상.
- ZIP/7z/RAR 압축 파일 내 이미지를 압축 해제 없이 직접 보기.
- 루비(후리가나)를 지원하는 아름다운 아오조라 문고(세로쓰기) 렌더링.
- 하드웨어 가속 이미지 렌더링으로 부드러운 성능 제공.
- 텍스트 파일의 스마트 인코딩 자동 감지.
- 원격 독서 및 감상을 위한 WebDAV 지원.
- 다크/라이트 모드를 지원하는 현대적인 Windows 11 스타일 인터페이스.
