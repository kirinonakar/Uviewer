產品名稱: Uviewer - 現代影像與小說閱讀器

簡短描述:
支援影像、文字、PDF 和 EPUB 的多功能檢視器，支援青空文庫(直書文字)與壓縮檔直接瀏覽。以 WinUI 3 打造的現代高效能體驗。

詳細描述:
Uviewer 是一款專為 Windows 打造的現代高效能整合檢視器，基於 WinUI 3 和 .NET 10 建構。它可順暢處理影像、文字、PDF 和 EPUB 文件，並提供壓縮檔內影像直接瀏覽、雙頁並排模式，以及完整的青空文庫直書文字渲染等進階功能。

Uviewer 也支援 Markdown、智慧編碼偵測和 WebDAV 連線，讓你的數位閱讀與瀏覽流程更有效率。

主要功能:

🖼️ 影像與 PDF 檢視
- 廣泛格式支援: PDF、JPEG、PNG、GIF、BMP、TIFF、ICO、SVG、WebP、AVIF、JXL 和動態 WebP。
- 高效能渲染: 使用 Win2D (Direct2D)，縮放和平移更加流暢。
- 智慧縮放: 支援自訂縮放比例(0.1x 到 10x)、符合視窗和實際大小。
- 工具: 銳利化濾鏡、雙頁並排模式、符合控制方向(支援由右至左導覽)和快速導覽浮層。

📝 文字與小說閱讀
- 進階文字引擎: 大型文字檔也能快速載入。
- 青空文庫支援: 原生渲染 Ruby 注音、強調、縮排和插圖。
- Markdown 支援: 自訂渲染表格與程式碼區塊。
- 智慧編碼偵測: 支援 UTF-8、EUC-KR、Shift-JIS 等。
- 直書模式: 以像素級版面精度提供完整的直書閱讀體驗。
- 導覽: 支援目錄擷取、PDF 書籤和跳至行。

📖 EPUB 閱讀器
- 完整 EPUB 支援: 基於頁面的導覽和章節追蹤。
- 目錄導覽: 快速在章節之間跳轉。
- 直書支援: 以傳統閱讀方式檢視 EPUB。

📂 檔案管理
- 壓縮檔支援: 無需解壓縮即可直接檢視 .zip、.rar、.7z 等壓縮檔中的影像。
- 整理功能: 管理收藏和最近檔案。
- WebDAV 支援: 直接存取遠端伺服器上的影像。

🌍 本地化
- 多語言支援: 英語、韓語、日語、簡體中文、繁體中文和越南語。

功能:
- 在一個應用程式中順暢檢視影像、文字、PDF 和 EPUB。
- 無需解壓縮即可直接檢視 ZIP/7z/RAR 壓縮檔中的影像。
- 支援 Ruby 注音的精美青空文庫(直書文字)渲染。
- 硬體加速影像渲染，提供流暢效能。
- 文字檔智慧編碼自動偵測。
- 支援 WebDAV 遠端閱讀與瀏覽。
- 現代 Windows 11 風格介面，支援深色/淺色模式。
